The new application icon, designed by [Katharina Hagemann](https://ktoons.org/), now comes in three variants for macOS 26 Tahoe, including Dark and Mono versions.

<img src="/images/posts/new-application-icon-for-mountain-duck.webp" alt="">

We decided to make it squircled so it fits more naturally with the new icon style. The changes are scheduled for [Mountain Duck 5.1](https://mountainduck.io/).

## New status bar icons

The status bar icon on macOS, and the tray icon on Windows, have also been updated. The aim is to make the main sync interface feel more at home on each platform while also showing sync progress more clearly with rotating arrows.
