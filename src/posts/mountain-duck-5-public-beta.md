Get a sneak peek at the upcoming [Mountain Duck 5](https://mountainduck.io/). The public beta introduces Integrated Connect Mode using native APIs on Windows and macOS to integrate directly in the file explorer, with no device driver installation or separate network mount required.

All major protocols remain supported, and [SMB](https://docs.mountainduck.io/) is added as a new option in version 5.

The new connect mode improves file write performance and file operations overall, bringing them much closer to the speed of local disk activity. Local storage is managed by the operating system, and the mount appears as a regular folder rather than a remote volume. That means it can be accessed when offline and managed with familiar controls in Finder and Windows Explorer.

Additional status indicators show synchronisation state next to file names, and uploads or downloads can display progress directly over the file icon. Windows Search and Spotlight on macOS can also be used to search for files.

To try the beta, choose Preferences → Update → Beta → Check for Update in the Mountain Duck status bar menu. The beta can be used with a valid version 4 registration key while testing.
